About Me

I’m a software developer and musician from Vancouver, Canada, and can be found most places online as iKevinY. The colourful bird pictured in my avatar is the lilac-breasted roller. Feel free to get in touch via email or Twitter!

This website is generated with Pelican, hosted on GitHub Pages, and open sourced on GitHub. The site’s theme is Pneumatic, using Eighties Dark for syntax highlighting. Articles are licensed under a Creative Commons Attribution license.

These are some of the applications, websites, and technologies that I use on a regular basis:

  • Sublime Text — source code editor with a wide variety of plugins available
  • GitHub — online Git-based project hosting
  • TotalTerminal — hotkey-activatable terminal overlay
  • f.lux — automatically adjust display colour temperature at night
  • Rainwave — video game music Internet radio station
  • Overcast — podcast player for iOS

Here some books that I highly recommend reading:

  • Code — Charles Petzold explains the design of computers and software in a manner accessible to even those without a background in computer science.
  • The Signal and the Noise — Nate Silver discusses how mathematical models created using probability and statistics are applied to real-world situations.
