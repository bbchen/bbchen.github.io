About

I’m an Earth Scientist from Honolulu, Hawaii, USA. I am passionate about science, technology, art, literature, and nature. Through this platform, I am sharing my thoughts, ideas, tips, and cool stuff to you and other readers.

This website is generated with Pelican, hosted on GitHub Pages, and open sourced on GitHub. The site’s theme is Pneumatic, using Eighties Dark for syntax highlighting. Articles are licensed under a Creative Commons Attribution license.

  • Sublime Text – source code editor with a wide variety of plugins available
  • GitHub – online Git-based project hosting
  • Omnifocus – Mac application adopting the GTD productivity principals
  • Zen Habits – Zen Habits is about finding simplicity and mindfulness in the daily chaos of our lives. It’s about clearing the clutter so we can focus on what’s important, create something amazing, find happiness.
