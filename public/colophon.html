Stack

This site is a DIY project to put in practice what I learn in my web development education journey.

I’m currently using a static site generator made with Go called Hugo. Hugo takes content written in Markdown and blends it with a set of templates that combine HTML and Go code, producing a static site that I then deploy to the public web via Github Pages.

For layout and styling, I am using a responsive framework called Skeleton, together with a custom style sheet to style some HTML elements, classes and IDs.

I keep track of changes to my site with Git, and store my source files in a Github remote repository.

I do most of my work from the command line, on a Linux terminal emulator called Git Bash that comes with Git for Windows.

For coding, I use either Sublime Text.
